
 Features:

Add Expenses: Log expenses with date, category, amount, and description.
Set Budgets: Set and update monthly budgets for categories.
View Reports: Track total spending and compare spending vs budget.

Tech Stack:
Python
Tkinter(for GUI)
SQLite (for database)


1. Clone the Repository:

   bash
   git clone https://github.com/yourusername/expense-tracker.git
   cd expense-tracker
   

2. Install Dependencies (if needed):

   bash
   pip install -r requirements.txt
   

3. Run the Application:

   bash
   python expense_tracker.py
   

Now, you can log expenses, set budgets, and view reports.
